# Monument(s)

Sur ce site, vous pourrez retrouver 2 monuments du Palais d'été de Pékin :
- **La Tour de l'encens bouddhique** : modélisée par Maxime Derènes
- **Le Bateau de marbre** : modélisé par Sébastien Chisiu

# Outil(s)

Afin de réaliser cette SAE, nous avons utilisé :
- **HTML, TailwindCSS, JS**
- **Mapbox 3D** : pour créer une carte du monde interactive
- **Google Model Viewer** : pour afficher les modèles 3D sur le site
- **Blender** : pour modéliser en 3D
- **Photoshop** : pour créer des textures
